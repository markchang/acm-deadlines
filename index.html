ACM Deadlines by Location

Shows the submission deadlines for the next six months of ACM conferences on a clickable map or in a list.

Check it out the map and the list

The goal is to provide a different interface to the ACM conference deadline calendar. Ideally, we would be able to

  • sort by date
  • sort by location
  • sort by conference

The data is scraped statically from http://campus.acm.org/public/cfpcal/Index.cfm

Last scrape: May 30, 2012
